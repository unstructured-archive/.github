# unstructured-archive

This organization is an archive of all major group projects we have worked on during our B. Tech in Computer Science. Not every member is involved in every project, so collaborators are listed individually for each repository.

---

## Projects

### fenris

**Fast Encrypted Networked Robust Information Storage**

A networked file storage system built in C++ with a focus on encryption and robustness. It provides a client-server architecture for securely storing and retrieving files over a network.

- Repository: [unstructured-archive/fenris](https://github.com/unstructured-archive/fenris)
- Language: C++
- Collaborators: [DarkPhoenix42](https://github.com/DarkPhoenix42), [v1bh475u](https://github.com/v1bh475u), [P4rth-B](https://github.com/P4rth-B)

---

### cielc

**An end-to-end compiler for the custom language Ciel**

A full compiler pipeline for a custom programming language called Ciel, implemented in C++. It covers lexing, parsing, semantic analysis, intermediate representation, and code generation targeting RISC-V.

- Repository: [unstructured-archive/cielc](https://github.com/unstructured-archive/cielc)
- Language: C++
- Collaborators: [DarkPhoenix42](https://github.com/DarkPhoenix42), [v1bh475u](https://github.com/v1bh475u), [G-Prototype](https://github.com/G-Prototype)

---

### bisqit

**Basic Interactive Simulator for Quantum Information and Technology**

A browser-based quantum circuit simulator built with TypeScript. It allows users to drag and drop quantum gates onto a circuit, run simulations, and visualize measurement results.

- Repository: [unstructured-archive/bisqit](https://github.com/unstructured-archive/bisqit)
- Language: TypeScript
- Collaborators: [DarkPhoenix42](https://github.com/DarkPhoenix42), [v1bh475u](https://github.com/v1bh475u), [G-Prototype](https://github.com/G-Prototype), [5h3rl0ckH0lm3s](https://github.com/5h3rl0ckH0lm3s)
